# Restricted-licence sources

Some in-scope instruments are authentic and obtained, but copyright-restricted, so their **full text is not
stored or redistributed in this corpus**. For each, the open layer carries the official citation, source URL,
content hash(es), and a short attributed excerpt; the full authentic file is held privately by the maintainer,
and its SHA-256 hash is published so anyone who independently obtains the same official file can verify
integrity. This follows the design-lock (JC-07) and the `redistribution: restricted` policy. The authoritative
record for each is at `authoritative/<id>/<version>/metadata.yaml` with `authoritative_status: restricted_withheld`
and **no stored text** — by design.
